Pong Tutorial project.

pong in win32 GUI

including:

* menu
* resizable screen
* simple pong ai
* score counter
* win function
* restart function

controls:
left player - 'w' up & 's' down.
right player - 'up-arrow' up & 'down-arrow' down.

single compile to win32_platform.cpp
